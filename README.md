# KTU OOP Report Generator

## Setup virtual enviroment
```shell
python -m venv venv
```

## Active virtual enviroment
```shell
source venv/bin/activate
```

## Deactivate virtual enviroment
```shell
deactivate
```

## Save requirements
```shell
pip freeze > requirements.txt
```
